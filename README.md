# Absolute Fitness

A modern, responsive website for **Absolute Fitness** - The Complete Health & Fitness Centre located in Jaipur, Rajasthan.

## About

Absolute Fitness is a premium gym offering personalized training programs including:

- **Cardio** - High-intensity cardio sessions
- **Zumba Dance** - Fun dance-based fitness classes
- **Yoga & Meditation** - Mind-body wellness programs
- **Weight Training** - Strength and muscle building
- **Workout** - General fitness programs

## Tech Stack

- **Next.js 14+** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations

## Pages

- `/` - Home
- `/about` - About the gym
- `/programs` - Membership packages
- `/testimonials` - Member success stories
- `/contact` - Contact form & location

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
